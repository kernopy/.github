# Kernopy
## An Industrial IoT Platform
Kernopy is an enterprise IoT platform designed to simplify complex IoT implementations.

## Features
- Advanced device management
- Support native MQTTs
- Supports Device OTA
- User friendly dashboards
- Build to scale
